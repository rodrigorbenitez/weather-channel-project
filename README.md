# Weather app application

A weather app that shows the current weather contitions form any country or location. It detect your local IP and returns your location. Then with https://openweathermap.org/ API returns data from any location you search.

## Getting Started

Start by either downloading the zip file or clone with HTTPS.

### Prerequisites

A modern web-browser like Safari, Chrome, Microsoft-Edge, Mozilla Firefox, etc.

### Running

Double-click the "index.html" file after unzipping the downloaded zip file.
